# Orvio Backend

Node.js + Express backend for the Orvio Project. Handles session events, inventory updates, and device communication.


## Tech Stack
- Node.js, Express  
- PostgreSQL (pg)  
- dotenv, cors, morgan  


## Setup
npm install
npm run dev
